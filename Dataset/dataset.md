# Dataset

## Source

Telco Customer Churn Dataset (Kaggle)

Link: https://www.kaggle.com/datasets/yeanzc/telco-customer-churn-ibm-dataset/data

## Description

This dataset contains customer-level information from a telecommunications company. It includes customer demographics, account information, subscribed services, charges, customer lifetime value (CLTV), and churn status.

## Data Includes

- Customer demographics
- Contract information
- Payment methods
- Internet and additional services
- Monthly charges
- Total charges
- Customer lifetime value 
- Churn status

## Usage

The dataset was imported into Google Bigquery, cleaned and transformed using SQL, and used to create Tableau dashboards analyzing customer churn patterns.
